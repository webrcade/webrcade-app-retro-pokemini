# webЯcade Application: Nintendo Pokémon Mini

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

The Nintendo Pokémon Mini application for [webЯcade](https://www.webrcade.com) is an emulator for the [Nintendo Pokémon Mini](https://en.wikipedia.org/wiki/Pok%C3%A9mon_Mini) handheld console.

The underlying emulator is based on the Libretro Pokémon Mini core.
